# SaaS-Export
ssm框架练手项目 货物代理运输 
## 平台主要功能
* 提供客户管理
* 权限管理
* 报关管理
* 运输管理
* 统计管理
为用户委托的运输任务的全部环节或者部分环节进行跟踪管理
Saas-Export 货贷云平台
